## Provider API

### JSON-RPC requests

## General

#### latest_upstream_block_height
[code]
----
"method": "call_api",
"params": {
    "api_id": "latest_upstream_block_height"
}
----

### latest_block_height
[code]
----
"method": "call_api",
"params": {
    "api_id": "latest_block_height",
    "api_params": {
        "transform_id": <transform_id: str>
    }
}
----

#### get_block
[code]
----
"method": "call_api",
"params": {
    "api_id": "get_block",
    "api_params": {
        "transform_id": <transform_id: str>
        "height": <height: int>
    }
}
----

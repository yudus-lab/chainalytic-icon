## Provider API

### JSON-RPC requests

#### latest_upstream_block_height
[code]
----
"method": "call_api",
"params": {
    "api_id": "latest_upstream_block_height"
}
----

#### contract_internal_transaction
[code]
----
"method": "call_api",
"params": {
    "api_id": "contract_internal_transaction",
    "api_params": {
        "address": "cx....",
        "size": 100 // Latest 100 txs
    }
}
----
